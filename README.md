# YOLO-street-view-traffic-sign-recognition
个人作业记录。  
空间分析课程的探索性实验，which means 做着玩的，which means 效果奇差。  
主要内容是利用北京路街景图像的交通标识标签训练YOLOv8（后面换成了11），再迁移学习到上下九街区进行检测。  
原题目后续还进行了对交通标识设置的合理性评估，由于那部分不是我做的内容所以删去了。  
  
YOLOv8引用：  
Jocher, G., Chaurasia, A., & Qiu, J. (2023). Ultralytics YOLO (Version 8.0.0) [Computer software]. https://github.com/ultralytics/ultralytics
